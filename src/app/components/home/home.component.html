EMPLOYEE TASK MANAGEMENT

Hi there this is Robins Ranjan

I've developed a task management web app using Angular and Firebase. Angular Material improves user experience, enabling smooth interactions like logging in, creating, updating, viewing, and deleting tasks. Firebase handles authentication via JWT tokens, ensuring secure task operations and user permissions.
